Categorias

Python

O que é Python e para que serve? O Python é uma linguagem de programação amplamente usada em aplicações da Web, desenvolvimento de software, ciência de dados e machine learning (ML). Os desenvolvedores usam o Python porque é eficiente e fácil de aprender e pode ser executada em muitas plataformas diferentes.

HTML

HTML (Linguagem de Marcação de HiperTexto) é o bloco de construção mais básico da web. Define o significado e a estrutura do conteúdo da web. Outras tecnologias além do HTML geralmente são usadas para descrever a aparência/apresentação (CSS) ou a funcionalidade/comportamento (JavaScript) de uma página da web.

Css

O CSS é uma linguagem de estilização que é aplicada aos elementos HTML em uma página da web. Basicamente, o CSS diz ao navegador como cada elemento HTML deve ser exibido na página. É possível usar o CSS direto no seu html, porém uma das formas mais indicadas para seu uso é criando um arquivo separado com a extensão “.
